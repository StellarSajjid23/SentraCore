# 🛡️ SentraCore:

> A Professional Python-Based SOC Monitoring, Log Analysis, and Threat Detection Toolkit

SentraCore is a Modular Security Toolkit Developed for Cybersecurity Students, SOC Analysts, and Defensive Security Learners.
This Project Combines Multiple Practical Log Monitoring and Threat Detection Tools into One Professional-Grade Python Repository.

---

## 🚀 Core Capabilities:

* ✅ Authentication Log Parsing and Monitoring
* ✅ Failed Login and Brute Force Detection
* ✅ Security Event Correlation
* ✅ Real-Time Threat Pattern Identification
* ✅ SIEM-Style Alert Logic
* ✅ Incident Severity Monitoring
* ✅ Suspicious Activity Summarization
* ✅ Modular SOC Automation Scripts

---

## Project Modules:

| Module           | Purpose                                                     |
| ---------------- | ----------------------------------------------------------- |
| BruteDemon       | Failed Login Monitoring and Brute Force Detection           |
| LogAnalyzer      | Authentication Log Parsing and Security Event Analysis      |
| MINI_SIEM_Engine | Correlated Threat Detection and SIEM-Style Alert Monitoring |
